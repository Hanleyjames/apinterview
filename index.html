Aesthetics Pro Interview Test

About this project:

This project is a short interview project for Aesthetics Pro. This project uses jquery, bootstrap, and Google Fonts. Primarily, I took advantage of this opportunity to work with jquery again, something I have not done in about two years.

Challenge #1: Basic Auth

Short Explenation: There is a heroku app hosting a JSON Object which requires an authorization header format with Basic as the token type, with username:password following.

Challenge #2: Object iteration and dynamic element creation/appending

Short Explenation: The same heroku application has a different endpoint on the same host, use the data to create dynamic elements

The test is to make sure you understand Object iteration and Dynamic element creation. The project uses jquery's each and append with string interpolation.
